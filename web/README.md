# Kiwi Web 🕷️

## Setup 🪛

1. Install `fnm` ([guide](https://github.com/Schniz/fnm))

1. Install the latest Node.js version:

   ```sh
   fnm install --latest --corepack-enabled
   fnm use <INSTALLED_VERSION>
   ```

1. Install dependencies with `pnpm install`

## Run Locally 🧸

```sh
pnpm dev
```

## Lint and Format 🧽

```sh
pnpm lint
pnpm format
```
